## What's solid

- Three-phase scan pipeline: walk -> parallel hash (rayon) -> batched DB commit
- Root enable/disable with `!` prefix in roots.conf
- Triage command with $EDITOR-based UX and .smritiignore apply
- Backup-audit command comparing roots for redundancy
- Streamable HTTP MCP server (chitta-rs pattern)
- SIGBUS fix: read-only commands use open_readonly (no migrations, query_only=ON)
- 80+ tests pass, release binary installed
